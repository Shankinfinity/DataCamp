# Predicting Health Insurance Charges

## Overview

In this project, I developed a regression model to predict individual health insurance charges using demographic and health-related information. After preparing the dataset by encoding categorical variables and scaling numerical features, I trained a Linear Regression model, evaluated its performance using cross-validation, and generated charge predictions for a separate validation dataset.

## Key Insights

* Prepared the dataset by encoding categorical variables and scaling numerical features.
* Built a Linear Regression model to predict health insurance charges.
* Evaluated the model using cross-validation with **R² Score** and **Mean Squared Error (MSE)**.
* Applied the trained model to estimate insurance charges for unseen validation data.
* Generated predicted insurance charges for each individual in the validation dataset.

## Skills Demonstrated

* Data preprocessing with Pandas
* Feature encoding and scaling
* Building regression models with scikit-learn
* Model evaluation using cross-validation, R² Score, and Mean Squared Error (MSE)
