# Exploring LEGO Star Wars Sets

## Overview

In this project, I analyzed LEGO set data to explore the popularity of licensed themes, with a particular focus on the Star Wars collection. Using Python and Pandas, I identified licensed LEGO sets, calculated the proportion of Star Wars sets among all licensed releases, and examined how the number of Star Wars sets changed over time to identify the year in which the theme became the most dominant.

## Key Insights

* Calculated the percentage of licensed LEGO sets that belong to the **Star Wars** theme.
* Filtered LEGO datasets to analyze only licensed product themes.
* Compared the number of licensed sets released for different themes over time.
* Identified the first year in which **Star Wars** became the most released licensed theme.

## Skills Demonstrated

* Data manipulation with Pandas
* Data cleaning and filtering
* Grouping and aggregating data
* Pivot table analysis
