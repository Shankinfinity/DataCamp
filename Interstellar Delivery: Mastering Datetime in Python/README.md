# Managing Rocket Delivery Dates with Python

## Overview

In this project, I developed a set of Python functions to manage dates and times for a fictional space logistics company. The project focuses on formatting Unix timestamps into readable dates, calculating rocket landing dates based on travel duration, and determining the number of days remaining until a scheduled delivery.

## Key Insights

* Converted Unix timestamps into human-readable date and time formats.
* Calculated rocket landing dates using launch dates and travel durations.
* Determined the number of days remaining until an expected cargo delivery.
* Created reusable functions to perform common date and time calculations.

## Skills Demonstrated

* Python function development
* Working with the `datetime` module
* Date and time calculations
* Formatting dates and timestamps
