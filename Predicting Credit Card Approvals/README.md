# Predicting Credit Card Application Approvals

## Overview

In this project, I developed a machine learning model to predict whether a credit card application would be approved. After cleaning the dataset by handling missing values, encoding categorical features, and scaling the data, I trained a Logistic Regression model and used GridSearchCV to identify the best hyperparameter settings. The final model was then evaluated on a separate test dataset.

## Key Insights

* Cleaned the dataset by handling missing values using appropriate imputation methods.
* Encoded categorical variables and standardized numerical features before model training.
* Built a Logistic Regression model to classify credit card applications.
* Used GridSearchCV with cross-validation to optimize the model's regularization parameter.
* Evaluated the final model using prediction accuracy and a confusion matrix.

## Skills Demonstrated

* Data preprocessing with Pandas
* Handling missing values
* Feature encoding and scaling
* Binary classification with Logistic Regression
* Hyperparameter tuning using GridSearchCV
