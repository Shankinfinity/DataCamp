# Clustering Penguins with K-Means

## Overview

In this project, I applied an unsupervised machine learning approach to group penguins based on their physical characteristics. After preparing and scaling the dataset, I used Principal Component Analysis (PCA) to visualize the data and K-Means clustering to identify natural groupings. Finally, I examined the average characteristics of each cluster to better understand the differences between them.

## Key Insights

* Prepared the dataset by selecting numerical features and standardizing their values.
* Reduced the dimensionality of the data using Principal Component Analysis (PCA) for visualization.
* Determined an appropriate number of clusters using the elbow method.
* Applied K-Means clustering to group penguins with similar physical characteristics.
* Compared the average measurements of each cluster to interpret the clustering results.

## Skills Demonstrated

* Data preprocessing with Pandas
* Feature scaling with StandardScaler
* Dimensionality reduction using PCA
* Unsupervised learning with K-Means clustering
* Cluster interpretation using summary statistics
