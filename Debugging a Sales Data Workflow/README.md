# Validating Sales Data Integrity

## Overview

In this project, I developed a data validation workflow to assess the integrity of a sales dataset using Python and Pandas. The process involved verifying the dataset structure, calculating statistical thresholds for daily sales totals, validating tax calculations, and identifying records that failed predefined integrity checks. The final workflow returns a processed dataset along with validation results.

## Key Insights

* Verified that the dataset contained the expected number of columns before processing.
* Calculated daily summary statistics to define acceptable ranges for transaction totals.
* Validated whether each transaction total fell within its expected range.
* Confirmed that the tax value for each transaction was correctly calculated based on quantity and unit price.
* Reported any records that failed the data integrity checks.

## Skills Demonstrated

* Data validation with Pandas
* Grouping and aggregating data
* Data integrity checking
* Creating validation rules using conditional logic
