# Exploring Global Fitness Trends with Google Trends

## Overview

In this project, I analyzed Google Trends data to explore how interest in fitness-related search terms has changed over time and across different countries. Using Python, I examined worldwide search trends for workout-related keywords, identified periods of peak interest, and compared the popularity of home workouts, gym workouts, and home gym searches in different regions.

## Key Insights

* Identified the year in which global interest in **"workout"** reached its peak.
* Compared search trends for **home workouts**, **gym workouts**, and **home gyms** over time.
* Determined which workout-related search term was the most popular in the most recent period.
* Compared overall workout interest across selected countries.
* Examined which country showed the highest interest in home workouts among selected Southeast Asian countries.

## Skills Demonstrated

* Data analysis with Pandas
* Working with time-series data
* Filtering and comparing datasets
* Data visualization with Matplotlib and Seaborn
