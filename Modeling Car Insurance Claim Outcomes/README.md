# Identifying the Best Predictor of Car Insurance Claims

## Overview

In this project, I analyzed a car insurance dataset to determine which individual customer feature best predicts whether an insurance claim will be filed. After cleaning the data and encoding categorical variables, I trained separate Logistic Regression models using one feature at a time and compared their predictive performance to identify the strongest predictor.

## Key Insights

* Cleaned the dataset by removing missing values and encoding categorical variables.
* Built individual Logistic Regression models using a single feature as the predictor.
* Evaluated the predictive accuracy of each feature independently.
* Ranked the features based on model accuracy to identify the strongest predictor of insurance claims.

## Skills Demonstrated

* Data preprocessing with Pandas
* Categorical data encoding
* Binary classification with Logistic Regression
* Model evaluation using accuracy
