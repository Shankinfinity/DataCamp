# Analyzing Flight Delays and Cancellations

## Overview

In this project, I analyzed flight data from 2022 to investigate patterns in delays and cancellations across different airlines and routes. I also explored whether weather conditions, specifically wind gusts, were associated with departure delays. Using Python, I processed the datasets, summarized key metrics, and created visualizations to compare airlines, routes, and weather-related trends.

## Key Insights

* Identified the routes with the highest average departure delays.
* Determined which routes experienced the greatest number of flight cancellations.
* Compared airlines based on their average departure delays and total cancellations.
* Examined whether wind gusts of **10 mph or greater** were associated with longer departure delays.

## Skills Demonstrated

* Data manipulation with Pandas
* Grouping and aggregating data
* Creating derived features
* Data visualization with Matplotlib
* Comparing trends across multiple categories
