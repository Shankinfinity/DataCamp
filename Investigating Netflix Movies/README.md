# Exploring Netflix Movies from the 1990s

## Overview

In this project, I analyzed a Netflix dataset to explore movies released during the 1990s. Using Python and Pandas, I filtered the dataset to focus on films from that decade, identified the most common movie duration, and examined the number of short action movies released during the same period.

## Key Insights

* Filtered the dataset to include only movies released between **1990 and 1999**.
* Identified the most common movie duration for films released during the 1990s.
* Analyzed action movies with a runtime of **less than 90 minutes**.
* Determined how many short action movies were released during the decade.

## Skills Demonstrated

* Data manipulation with Pandas
* Filtering datasets using multiple conditions
* Descriptive statistical analysis
* Extracting insights from real-world data
