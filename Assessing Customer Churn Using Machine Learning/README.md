# Predicting Telecom Customer Churn

## Overview

In this project, I built a machine learning model to predict customer churn using telecom customer data from India. After merging demographic and usage datasets, I prepared the data by encoding categorical variables, scaling numerical features, and splitting the data into training and testing sets. I then trained and evaluated both Logistic Regression and Random Forest models to compare their performance in predicting customer churn.

## Key Insights

* Combined customer demographic and usage data into a single dataset for analysis.
* Calculated the proportion of customers who had churned.
* Prepared the data by encoding categorical variables and scaling numerical features.
* Compared the performance of Logistic Regression and Random Forest using accuracy, confusion matrices, and classification reports.
* Identified the model that achieved the better predictive performance on the test data.

## Skills Demonstrated

* Data preprocessing with Pandas
* Feature encoding and scaling
* Training classification models with scikit-learn
* Model evaluation using confusion matrices and classification reports
