# Optimizing a Data Science Job Dataset

## Overview

In this project, I optimized a data science job dataset by reducing its memory usage through appropriate data type conversions. The dataset was transformed by assigning efficient numerical and categorical data types, defining ordered categorical variables where applicable, and filtering records based on specific experience and company size criteria. The result is a more memory-efficient dataset that is better suited for further analysis and machine learning tasks.

## Key Insights

* Reduced memory usage by converting columns to more efficient data types.
* Transformed nominal and ordinal variables into categorical data types.
* Defined meaningful category orders for education level, experience, company size, and previous job changes.
* Filtered the dataset to include professionals with **10 or more years of experience** working at companies with **1,000 or more employees**.

## Skills Demonstrated

* Data preprocessing with Pandas
* Data type optimization
* Working with categorical and ordinal variables
* Filtering datasets based on multiple conditions
