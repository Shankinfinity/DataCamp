# Exploring New York City Airbnb Listings

## Overview

In this project, I analyzed Airbnb listing data from New York City by combining information from multiple datasets. The analysis focused on identifying the range of review dates, determining the number of private room listings, and calculating the average listing price. Using Python and Pandas, I cleaned the data, standardized text values, and summarized the results to answer these key business questions.

## Key Insights

* Identified the earliest and most recent review dates in the dataset.
* Standardized room type categories to ensure consistent analysis.
* Calculated the total number of listings classified as **private rooms**.
* Converted listing prices into numerical values and computed the average price across all listings.

## Skills Demonstrated

* Data cleaning with Pandas
* Merging information from multiple datasets
* Data type conversion
* Data aggregation and summary statistics
