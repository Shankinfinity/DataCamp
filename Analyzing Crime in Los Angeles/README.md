# Analyzing Crime in Los Angeles

## Overview

In this project, I analyzed crime data from Los Angeles using Python to answer a few specific questions about crime patterns. The analysis focused on identifying the hour with the highest number of reported crimes, determining which area experienced the most crimes during the night, and examining the distribution of victims across different age groups.

## Key Insights

* Identified the hour of the day with the highest number of reported crimes.
* Determined which Los Angeles area recorded the most crimes between **10 PM and 3 AM**.
* Grouped victims into age categories to understand how crime reports were distributed across different age groups.

## Skills Demonstrated

* Data manipulation with Pandas
* Creating new features from existing data
* Filtering data using multiple conditions
* Grouping and counting records to answer analytical questions
