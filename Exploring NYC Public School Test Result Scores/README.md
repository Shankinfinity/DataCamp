# Analyzing SAT Performance in New York City Schools

## Overview

In this project, I analyzed SAT performance data from New York City public schools using Python and Pandas. The analysis focused on identifying schools with strong mathematics performance, ranking schools by their combined SAT scores, and comparing overall SAT performance across different boroughs using summary statistics.

## Key Insights

* Identified schools with an average mathematics score of **640 or higher**.
* Calculated the total SAT score for each school by combining mathematics, reading, and writing scores.
* Ranked the top-performing schools based on their total SAT scores.
* Compared SAT performance across boroughs by calculating the number of schools, average SAT score, and standard deviation for each borough.

## Skills Demonstrated

* Data manipulation with Pandas
* Creating derived features
* Sorting and filtering datasets
* Grouping and aggregating data
