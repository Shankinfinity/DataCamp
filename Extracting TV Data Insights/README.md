# Exploring Super Bowl Trends

## Overview

In this project, I analyzed historical Super Bowl data to explore trends in television viewership, game competitiveness, and halftime performances. Using Python and Pandas, I examined changes in U.S. TV audiences over time, investigated the distribution of winning margins, and identified the musician who performed the greatest number of halftime songs.

## Key Insights

* Examined how average U.S. television viewership changed across Super Bowl events.
* Analyzed the distribution of point differences to assess how competitive Super Bowl games have been.
* Counted the number of games decided by more than 40 points.
* Identified the halftime performer who sang the highest number of songs across all appearances.

## Skills Demonstrated

* Data analysis with Pandas
* Data visualization with Matplotlib
* Grouping and aggregating data
* Exploratory data analysis
