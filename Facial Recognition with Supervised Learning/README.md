# Facial Recognition with Machine Learning

## Overview

In this project, I developed a machine learning pipeline to distinguish images of Arnold Schwarzenegger from other individuals using facial image data from the Labeled Faces in the Wild (LFW) dataset. I trained and compared multiple classification models, optimized their hyperparameters using cross-validation, and evaluated the best-performing model on a separate test dataset.

## Key Insights

* Built preprocessing pipelines that combined feature scaling with machine learning classifiers.
* Compared the performance of Logistic Regression, K-Nearest Neighbors, and Decision Tree models.
* Used grid search with cross-validation to identify the best hyperparameters for each model.
* Evaluated the best-performing classifier using accuracy, precision, recall, F1-score, and a confusion matrix.
* Selected the model that achieved the highest cross-validation performance for facial recognition.

## Skills Demonstrated

* Building machine learning pipelines with scikit-learn
* Hyperparameter tuning using GridSearchCV
* Cross-validation for model selection
* Classification model evaluation using standard performance metrics
