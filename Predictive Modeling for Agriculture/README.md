# Identifying the Best Soil Feature for Crop Classification

## Overview

In this project, I developed a machine learning model to determine which soil property is the strongest predictor of crop type. Using agricultural data containing soil nutrient levels and pH values, I trained a Random Forest Classifier to classify different crops and evaluated feature importance to identify the variable that contributed most to the model's predictions.

## Key Insights

* Prepared the dataset by selecting soil nutrient and pH features for classification.
* Split the data into training and testing sets for model evaluation.
* Trained a Random Forest Classifier to predict crop type.
* Calculated feature importance scores to measure the contribution of each soil property.
* Identified the single most predictive feature for distinguishing between crop types.

## Skills Demonstrated

* Data preprocessing with Pandas
* Building classification models with scikit-learn
* Random Forest classification
* Feature importance analysis
