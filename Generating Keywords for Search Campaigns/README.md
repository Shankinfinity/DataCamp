# Generating Search Engine Marketing Keywords

## Overview

In this project, I developed a Python script to automatically generate search engine marketing (SEM) keywords for an online furniture retailer. By combining product names with common search terms, I created a structured list of keywords that can be used in advertising campaigns. The generated keywords were organized into a tabular format and exported as a CSV file for further use.

## Key Insights

* Generated keyword combinations by pairing furniture products with common search terms.
* Created both **"modifier + product"** and **"product + modifier"** keyword variations.
* Organized the generated keywords into advertising groups with campaign information.
* Assigned each keyword an **Exact Match** criterion for search advertising.
* Exported the complete keyword list as a CSV file for use in marketing campaigns.

## Skills Demonstrated

* Python automation
* Data manipulation with Pandas
* String manipulation
* Exporting structured data to CSV
