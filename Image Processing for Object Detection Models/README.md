# Preprocessing Wildlife Camera Trap Images

## Overview

In this project, I developed an image preprocessing pipeline to prepare wildlife camera trap images for machine learning applications. Using Python and scikit-image, I created a reusable function that imports multiple images, applies common image transformations, and returns the processed images in a consistent format suitable for further analysis or model training.

## Key Insights

* Developed a reusable function to process multiple images in a single workflow.
* Resized images to a consistent resolution for downstream applications.
* Applied image rotation to support image transformation.
* Enhanced image contrast using adaptive histogram equalization.
* Generated processed images that are ready for computer vision tasks.

## Skills Demonstrated

* Image preprocessing with scikit-image
* Image manipulation using Matplotlib
* Function development in Python
* Batch processing of image datasets
