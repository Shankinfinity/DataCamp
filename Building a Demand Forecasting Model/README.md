# Forecasting Online Retail Sales with PySpark

## Overview

In this project, I built a sales forecasting model using an online retail dataset and PySpark. After preparing the transactional data, I engineered time-based features, split the dataset into training and testing periods, and trained a Random Forest Regressor to predict product quantities sold. The model was then evaluated to assess its forecasting performance on unseen data.

## Key Insights

* Prepared daily sales data by extracting date-related features from transaction records.
* Split the dataset into training and testing sets based on a specified date.
* Built a Random Forest regression model using PySpark ML.
* Evaluated the model using Mean Absolute Error (MAE).
* Generated weekly sales forecasts from the model's predictions.

## Skills Demonstrated

* Data preprocessing with PySpark
* Feature engineering for time-series data
* Building regression models with PySpark ML
* Model evaluation using Mean Absolute Error (MAE)
