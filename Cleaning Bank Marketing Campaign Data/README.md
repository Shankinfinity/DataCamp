# Cleaning and Restructuring Bank Marketing Data

## Overview

In this project, I cleaned and reorganized a bank marketing dataset into multiple structured tables using Python and Pandas. The data was reformatted by selecting relevant columns, standardizing text values, converting variables to appropriate data types, and separating the dataset into client, campaign, and economic information. The final cleaned datasets were exported as CSV files for future analysis.

## Key Insights

* Organized the original dataset into three separate tables: client, campaign, and economics.
* Standardized categorical values by replacing inconsistent formatting.
* Converted categorical variables into appropriate Boolean data types.
* Exported the cleaned datasets as individual CSV files for downstream use.

## Skills Demonstrated

* Data cleaning with Pandas
* Selecting and restructuring datasets
* Data type conversion
* Exporting processed data to CSV files
