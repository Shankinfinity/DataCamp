# Analyzing Electric Vehicle Sales and Charging Infrastructure

## Overview

In this project, I analyzed electric vehicle (EV) sales alongside public and private charging infrastructure data in the United States. Using Python, I combined multiple datasets to explore how EV adoption changed over time and examined whether the growth in charging stations kept pace with increasing vehicle sales.

## Key Insights

* Calculated the total number of electric vehicles sold in **2018**.
* Combined public and private charging infrastructure datasets into a single dataset for analysis.
* Compared trends in EV sales with the growth of public charging station locations and private charging ports.
* Examined changes in charging infrastructure between **2015 and 2018** to evaluate how it evolved alongside increasing EV adoption.

## Skills Demonstrated

* Data manipulation with Pandas
* Merging multiple datasets
* Time-series data analysis
* Data visualization with Matplotlib and Seaborn
