# Discovering Topics in Mobile App Reviews

## Overview

In this project, I analyzed negative mobile app reviews to identify common user concerns using natural language processing techniques. After preprocessing the review text by tokenizing, removing stopwords, and lemmatizing words, I transformed the text into TF-IDF features and applied K-Means clustering to group similar reviews into distinct topics.

## Key Insights

* Filtered the dataset to focus on negative app reviews.
* Preprocessed review text through tokenization, stopword removal, and lemmatization.
* Converted text into numerical features using TF-IDF vectorization.
* Applied K-Means clustering to group reviews with similar content.
* Identified the most representative keywords for each cluster to interpret the underlying topics.

## Skills Demonstrated

* Text preprocessing with NLTK
* TF-IDF feature extraction
* Unsupervised learning with K-Means clustering
* Topic interpretation using keyword analysis
