# Analyzing Sleep Health and Lifestyle Data

## Overview

In this project, I analyzed sleep and lifestyle data collected from a sleep tracking application to investigate how sleep patterns vary across different occupations and health characteristics. Using Python and Pandas, I examined average sleep duration and sleep quality by occupation and explored the relationship between BMI categories and the prevalence of insomnia.

## Key Insights

* Identified the occupation with the lowest average sleep duration.
* Determined the occupation with the lowest average sleep quality.
* Compared whether the occupations with the lowest sleep duration and sleep quality were the same.
* Calculated the proportion of individuals with insomnia within each BMI category.

## Skills Demonstrated

* Data manipulation with Pandas
* Grouping and aggregating data
* Descriptive statistical analysis
* Working with dictionaries to summarize results
