# Predicting DVD Rental Duration

## Overview

In this project, I developed a machine learning model to predict the number of days a customer will rent a DVD. After preparing the dataset by creating new features from rental dates and extracting information from movie special features, I trained and compared multiple regression models to identify the one that achieved the lowest prediction error.

## Key Insights

* Created a target variable representing the number of rental days from the rental and return dates.
* Engineered new features from the movie's special features, including indicators for deleted scenes and behind-the-scenes content.
* Split the dataset into training and testing sets for model evaluation.
* Compared the performance of Linear Regression, Ridge Regression, and Random Forest Regression models.
* Selected the model with the lowest Mean Squared Error (MSE) as the final predictor.

## Skills Demonstrated

* Data preprocessing with Pandas
* Feature engineering
* Building regression models with scikit-learn
* Model comparison and evaluation using Mean Squared Error (MSE)
